# IMDb Dashboard

---
## Dashboard Overview

This Power BI dashboard analyzes IMDb movie data to uncover trends in ratings, revenue, audience votes, metascore, and yearly movie releases.

It presents movie insights through interactive visualizations, KPI cards, and filter-based analysis.

## Dataset

The project uses an IMDb movie dataset containing movie-level information such as:

* Movie Title
* Year
* Genre
* Director
* Actors
* Rating
* Votes
* Revenue (Millions)
* Metascore
* Runtime (Minutes)

---
## Dashboard Highlights

* Average Revenue KPI
* Average Votes KPI
* Average Metascore KPI
* Movies Released by Year
* Votes Analysis by Movie
* Revenue Trend by Year
* Metascore Comparison by Year
* Interactive Year Filter

---
## Key Insights

* Audience engagement varies significantly across movies and years.
* Revenue trends fluctuate over time.
* Some years show stronger critical reception through metascores.
* A small number of movies receive the highest share of audience votes.
